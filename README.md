# Modelo-Predictivo-Retorno-Bitcoin
## Objetivos del análisis: 
### Objetivo general: 
Desarrollar un modelo para estimar el precio futuro del Bitcoin en función de sus datos 
históricos, con el objetivo de proporcionar estrategias. 
### Objetivos específicos: 
1. Analizar los datos históricos del Bitcoin para identificar patrones significativos. 
2. Entrenar un modelo utilizando técnicas de regresión para proyectar el precio 
futuro del Bitcoin. 
3. Crear gráficas que muestren las proyecciones de los precios del Bitcoin para 
facilitar su entendimiento. 
## Fuentes de datos: 
Para este proyecto, se usará el dataset “Bitcoin Historical Data”. Este dataset contiene los 
datos históricos del precio del Bitcoin, desde 2012 hasta la fecha. Esto es crucial para el 
análisis predictivo, ya que brinda una buena base de datos históricos que se usarán para 
el entrenamiento del modelo. 
### Enlace: [Bitcoin Historical Data ](https://www.kaggle.com/datasets/mczielinski/bitcoin-historical-data?resource=download)
## Alcance preliminar del proyecto: 
Este análisis se centrará únicamente en la predicción del precio del Bitcoin, utilizando 
sus datos históricos. No se incluirán análisis de otras criptomonedas, ni causas de la 
volatilidad del precio ni se considerarán factores como regulaciones externas o noticias 
globales. 
Se asume que los datos históricos son representativos del comportamiento futuro del 
mercado y que no habrá eventos externos que puedan afectarlo.
